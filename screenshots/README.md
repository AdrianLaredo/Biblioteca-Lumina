# Capturas de Pantalla - Biblioteca Lumina

Esta carpeta contiene las capturas de pantalla del sistema para documentación.

## Capturas Requeridas

Por favor, agrega las siguientes capturas de pantalla con los nombres especificados:

### Generales
1. **login.png** - Pantalla de inicio de sesión

### Rol Administrador
2. **dashboard-admin.png** - Dashboard del administrador con estadísticas
3. **gestion-libros.png** - Pantalla de gestión de libros (agregar/editar/eliminar)
4. **gestion-usuarios.png** - Pantalla de administración de usuarios
5. **reportes.png** - Visualización de reportes y estadísticas del sistema

### Rol Docente
6. **dashboard-docente.png** - Dashboard personalizado para docentes
7. **catalogo.png** - Vista del catálogo de libros
8. **mis-prestamos.png** - Vista de préstamos activos del usuario
9. **historial.png** - Historial de préstamos del usuario

### General (Docente/Estudiante)
10. **solicitar-prestamo.png** - Interfaz para solicitar préstamo de libros

### Rol Estudiante
11. **dashboard-estudiante.png** - Dashboard personalizado para estudiantes

## Instrucciones para Capturar

1. Iniciar sesión con cada usuario de prueba:
   - Admin: admin / admin123
   - Docente: amartinez / docente123
   - Estudiante: jperez / estudiante123

2. Navegar por las diferentes secciones

3. Capturar pantallas completas o de la sección relevante

4. Guardar con los nombres especificados arriba

5. Formato recomendado: PNG o JPG

## Dimensiones Recomendadas

- Ancho: mínimo 1200px
- Mantener relación de aspecto original
- Calidad: alta (para buena visualización en documentación)
